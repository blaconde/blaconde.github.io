Featured Project

Health & Lifestyle Dashboard

Interactive Power BI dashboard designed to present lifestyle, wellbeing, and activity trends in a clear, visual, and user-friendly way.

Power BI Power Query Data Visualization Dashboard Design

Project Overview

This project was created as a Power BI dashboard concept focused on presenting health and lifestyle data in a clear and engaging format. The goal was to combine multiple indicators in one readable report and make trend analysis easy for the user.

The dashboard brings together activity, sleep, stress, hydration, mood, and training-related metrics, allowing the user to explore patterns over time and compare selected categories through filters and interactive visuals.

My Focus

  • Clear visual hierarchy and readable layout
  • Consistent dashboard styling and intuitive navigation
  • Presenting multiple KPIs without overwhelming the user
  • Interactive filtering for easier data exploration
  • Combining overview metrics with more detailed visual analysis

What the Dashboard Includes

  • KPI cards summarizing the most important metrics
  • Trend charts showing changes over time
  • Comparisons across training intensity and wellness factors
  • Mood distribution overview
  • Filters for date, gender, age band, mood, and training intensity

Why This Project Matters

This project demonstrates my approach to designing reports that are both analytical and accessible. I focused not only on the technical side of building the dashboard, but also on the user experience — making sure the report is readable, structured, and visually coherent.

It is a good example of how I translate data into a format that supports quick understanding and better decision-making.
